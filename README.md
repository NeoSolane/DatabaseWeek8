# Task Manager API
A simple Restful API built using Node.js + Express, and MYSQL for managing tasks and users.The project demonstrates full CRUD operations
## Objectives
-create,read,update,and delete tasks.
-create and view users.
## Techologies used
- Node.js
- Express
- MySQL
- Dotenv
- Body-parser
  ## How to run/view the project
  -start your mysql
  -create databes and tables
  -run the server
  -test the API using tools like postman or any tool used for API testing
